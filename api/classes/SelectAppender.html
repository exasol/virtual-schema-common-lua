selectappender.workaround_group_by_integer(group_by_criteria) line 113
Replace an unsupported expression in a GROUP BY clause with a supported one or return it unchanged.

This replaces numeric literals with the corresponding string value, as Exasol interprets GROUP BY <integer-constant> as column number — which is not what the user intended. Also, please note that GROUP BY <constant> always leads to grouping with a single group, regardless of the actual value of the constant (except for FALSE, which is reserved).

Parameters:

  • group_by_criteria the original GROUP BY expression

Returns:

    a new, alternative expression or the original expression if no replacement is necessary
