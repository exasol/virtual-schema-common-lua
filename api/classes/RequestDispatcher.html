Class RequestDispatcher

This class dispatches Virtual Schema requests to a Virtual Schema adapter.

It is independent of the use case of the VS adapter and offers functionality that each Virtual Schema needs, like JSON decoding and encoding and setting up remote logging. To use the dispatcher, you need to inject the concrete adapter the dispatcher should send the prepared requests to.
