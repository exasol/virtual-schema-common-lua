Common base library for Virtual Schemas written in Lua that run on Exasol

Please visit the project homepage on GitHub for detailed information.

Modules

exasol.vscl.text This module contains basic string manipulation methods not included in the Lua standard library.
exasol.validator Validators for common input such as user names or port numbers.

Classes

AbstractVirtualSchemaAdapter This class implements an abstract base adapter with common behavior for some of the request callback functions.
AdapterProperties This class abstracts access to the user-defined properties of the Virtual Schema.
ImportQueryBuilder Builder for an IMPORT query that wraps push-down query
Query This class implements an abstraction for a query string including its tokens.
QueryRenderer Renderer for SQL queries.
RequestDispatcher This class dispatches Virtual Schema requests to a Virtual Schema adapter.
AbstractQueryRenderer This class is the abstract base class of all query renderers.
AggregateFunctionAppender Appender for aggregate functions in an SQL statement.
ExpressionAppender Appender for value expressions in a SQL query.
ImportAppender Appender that can add top-level elements of a SELECT statement (or sub-select).
ScalarFunctionAppender Appender for scalar functions in an SQL statement.
SelectAppender Appender that can add top-level elements of a SELECT statement (or sub-select).
